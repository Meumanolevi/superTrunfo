Como Jogar — Passo a Passo

  1. Comece a partida: abra Jogar e pressione "Iniciar" para distribuir as cartas iniciais.
  2. Compra: use o botão "Comprar" para adicionar cartas à sua mão (máx. 5 cartas visíveis).
  3. Jogar uma carta: arraste uma carta da sua mão e solte no seu slot (à sua frente). Ao jogar, escolha qual atributo (Fé, Coragem, Influência) deseja usar para a batalha.
  4. Contra-ataque: o oponente jogará uma carta automaticamente; se você já escolheu um atributo, o oponente tenta responder adequadamente.
  5. Comparação: o atributo escolhido é comparado entre as cartas. Quem vencer causa dano ao outro jogador. Em empate, ninguém sofre dano.
  6. Término: o jogo termina quando a vida de um jogador chega a 0. Use "Iniciar" para reiniciar a partida.

Dicas Visuais

  • Passe o mouse sobre uma carta para ver seus atributos de forma detalhada sem poluir a frente da carta.
  • Mantenha cartas de alto valor para quando o atributo selecionado for mais vantajoso.
  • Use o botão "Finalizar Turno" (ou pressione T) para forçar o oponente a jogar se quiser acelerar a partida.

Termos

Fé: representa poder místico; Coragem: força física/impacto; Influência: capacidade estratégica/social. O "Poder Total" é a soma desses três valores.

Manual rápido — estilo minimalista para manter a experiência coesa.
